# ikonfx

The Ikonli Icons browser made for JavaFX developers to boost productivity

### build and run the application using gradle

#### prerequisites

1. Java 14
2. Gradle 6+

```bash

 gradle jlink
 cd build/image/bin
./ikonfx

```
